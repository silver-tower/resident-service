== 사용자 등록

새로운 사용자를 등록합니다.

[discrete]
==== 요청

include::{snippets}/users/curl-request.adoc[]

include::{snippets}/users/request-body.adoc[]

==== Request Body 필드

|====
|속성        |설명                                   |필수 여부

|`name`      |사용자의 이름                           |O
|`birthDate` |사용자의 생년월일 (YYYY-MM-DD 형식)     |O
|`phoneNumber` |사용자 전화번호                        |O
|`email`     |사용자 이메일 주소                      |O
|`gender`    |성별 (M 또는 F)                        |O
|====

==== Request Body (예시)

[source,json]
----
{
  "name": "홍길동",
  "birthDate": "1985-05-20",
  "phoneNumber": "01012345678",
  "email": "test@example.com",
  "gender": "M"
}
----

==== 에러 응답

요청이 실패한 경우, 다음과 같은 응답이 반환됩니다:

.Error Body (예시)
[source,json]
----
{
  "success": false,
  "timestamp": "2025-06-26T12:34:56",
  "error": {
    "code": "U4000",
    "message": "필수 필드 입력이 누락되었습니다."
  },
  "uuid": "abcd-1234-efgh-5678"
}
----
